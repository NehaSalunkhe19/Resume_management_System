# Resume Management System

A web-based **Resume Management System** developed using **Django**.  
This project allows users to upload resumes and apply for jobs, while admins can efficiently manage applications.

---

##  User Features
- User registration and login
- Upload and manage resume
- Apply for jobs
- View application status (Pending / Approved / Rejected)
- Receive email confirmation after registration and job application

---

## Admin Features
- Admin dashboard
- View all job applications
- Approve or reject applications
- Manage users and job postings
- Track application status

---

## Technologies Used
- **Backend:** Django  
- **Frontend:** HTML, CSS  
- **Database:** MySQL  
- **Tools:** Git, GitHub  
- **Email Service:** Django Email Backend  

---

## Project Modules
- User Authentication
- Resume Upload Module
- Job Posting Module
- Job Application Module
- Application Status Tracking
- Email Notification System
- Admin Dashboard

---

## ⚙️ How to Run the Project

1. Clone the repository  
2. Create a virtual environment  
3. Activate the virtual environment  
4. Install required dependencies  
5. Configure database settings in `settings.py`  
6. Apply migrations  
7. Create a superuser  
8. Run the development server  
9. Open browser and access the project  
10. Login as admin or user  
11. Start using the application
